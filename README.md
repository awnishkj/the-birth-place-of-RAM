## 🚀 Quick Start Guide

<div align="center">

### 📋 Prerequisites
![Node.js](https://img.shields.io/badge/Node.js-v18+-339933?style=flat-square&logo=node.js&logoColor=white)
![npm](https://img.shields.io/badge/npm-latest-CB3837?style=flat-square&logo=npm&logoColor=white)

</div>

### 📥 Installation

<details>
<summary><b>🔽 Click to expand installation steps</b></summary>

```bash
# 1️⃣ Clone the sacred repository
git clone https://github.com/awnishkj/the-birth-place-of-RAM.git
cd the-birth-place-of-RAM

# 2️⃣ Install Frontend Dependencies
cd frontend
npm install

# 3️⃣ Install Backend Dependencies  
cd ../server
npm install

# 4️⃣ Environment Setup
# Create .env file in server directory
echo "EMAIL_USER=your-email@gmail.com" > server/.env
echo "EMAIL_PASS=your-app-password" >> server/.env
```

</details>

### 🏃‍♂️ Running the Application

<table>
<tr>
<td width="50%">

#### 🖥️ **Development Mode**
```bash
# Terminal 1 - Backend
cd server
npm run dev
# 🌐 Server: http://localhost:5000

# Terminal 2 - Frontend  
cd frontend
npm run dev
# 🌐 Frontend: http://localhost:5173
```

</td>
<td width="50%">

#### 🚀 **Production Mode**
```bash
# Build Frontend
cd frontend
npm run build

# Start Production Server
cd ../server
npm start
```

</td>
</tr>
</table>
